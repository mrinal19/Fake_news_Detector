# 🧠 AI Fake News Detector

🚀 A real-time AI-powered web application that analyzes news content and predicts whether it is **Fake 🟥 or Real 🟩** using Machine Learning.

---

## 🌐 Live Demo

👉 https://fake-news-detector-7hym.onrender.com

---

## ✨ Features

* 🔍 Real-time fake news detection
* ⚡ Fast prediction using trained ML model
* 🎯 Clean and interactive UI
* 🧠 NLP-based text analysis
* 🌐 Deployed and accessible online

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn, TF-IDF
* **Deployment:** Render

---

## 📂 Project Structure

```
fake_news_detector/

│   └── index.html
├── static/
│   └── style.css

```

---

## ⚙️ How It Works

1. User inputs news content
2. Text is preprocessed and vectorized
3. ML model predicts authenticity
4. Result is displayed instantly

---

## 🚀 Run Locally

```bash
git clone https://github.com/mrinal19/Fake_news_Detector.git
cd Fake_news_Detector
pip install -r requirements.txt
python app.py
```
